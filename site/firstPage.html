Additionally, it's very important to note the 4 types of links (see below)!

The 4 types of links:

  1. Absolute Reference: links to a fully formed URL and needs http or https
  2. Relative Reference: links to a file or a folder. Also used to call an ID tag
  3. Internal Links: links that go from one page on a domain to a different page on the same domain.
  4. Graphical Links: links that are "clickable" icons or images. The clickable component doesn't need to be text.
